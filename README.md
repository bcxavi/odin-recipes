# Odin Recipes

## Description
This project is a basic recipe website built as part of The Odin Project. It includes a main index page with links to several recipe pages, each displaying ingredients and steps.

The website uses a shared CSS file to provide a consistent layout and styling across all pages.

## Features
- Main index page with links to recipes
- Individual recipe pages
- Reusable card layout
- Styled navigation and content lists
- Responsive-friendly layout

## Skills Demonstrated
- Structuring HTML pages
- Using links, images, and lists
- Working with relative file paths
- Applying CSS for layout and styling
- Reusing styles across multiple pages
- Separating navigation and content components

## Technologies Used
- HTML
- CSS
